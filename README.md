# Proyecto Campaña de Renta

Este es el proyecto base para la página de Campaña de Renta. Se ha construido un scaffolding con HTML, CSS y JavaScript nativo, implementando un diseño moderno, estéticamente agradable, centrado en conversiones y con animaciones sutiles.

## Estructura de Carpetas

- `/assets/logo`: Directorio para los logos de la campaña y marca.
- `/assets/images`: Directorio para las imágenes, ilustraciones y fondos de la web.
- `/pages/renta`: Contiene los archivos principales de la aplicación web de renta.
  - `index.html`: Estructura semántica de la página web.
  - `style.css`: Estilos premium utilizando variables CSS, glassmorphism y diseño responsivo.
  - `main.js`: Lógica de interacción, animaciones sutiles y comportamiento del DOM.
- `/index.html`: Archivo en la raíz del proyecto que se encarga de redirigir al usuario al `index.html` dentro de `/pages/renta`.

## Uso y Desarrollo

1. Abre el archivo `/index.html` de la raíz en cualquier navegador moderno.
2. Automáticamente serás redirigido a `/pages/renta/index.html`.
3. Edita los archivos dentro de `/pages/renta` para personalizar la página.
